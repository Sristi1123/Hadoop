🔹 What is -expunge?
hdfs dfs -expunge

👉 It permanently deletes files that are currently in the Trash.

🧠 How Trash works in HDFS

When you delete a file using:

hdfs dfs -rm /file.txt

❌ It is NOT immediately deleted
✔ It goes to a hidden Trash folder:

/user/<username>/.Trash/
🔥 Role of -expunge
hdfs dfs -expunge

✔ Clears the Trash
✔ Permanently removes deleted files
✔ Frees storage space

📌 Example Flow
Step 1: Delete a file
hdfs dfs -rm /BACKUP/file1.txt

👉 File goes to Trash

Step 2: Empty Trash
hdfs dfs -expunge

👉 File is now permanently deleted

⚠️ Important Points
❗ After -expunge, recovery is NOT possible
❗ Works only if Trash is enabled (fs.trash.interval > 0)
❗ No confirmation prompt → use carefully
